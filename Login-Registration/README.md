# Login & Registration System (PHP)


A simple authentication system built with PHP and MySQL.


## Features
- User registration
- Secure login
- Password hashing
- Sessions
- Logout system


## Usage
1. Create database `login_system`
2. Import the users table SQL
3. Update db.php credentials
4. Run project via localhost
